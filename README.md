# How to use ?

You need install all package:

```
npm install
```

Cd to client

```
cd client
npm install
```

You must import .env, we provide to folder Final-Web-Banking-App.

Because we use mongose atlas so you don't need install mongose compass.

Data will in the cloud.

Run application

```
cd ..
npm start
```
